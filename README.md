## 安装django
1. 使用python3
2. 安装venv，参考：https://docs.djangoproject.com/en/3.1/intro/contributing/
3. 进入venv，安装django(可安装指定版本): pip3 install Django==2.2.0

## 修改了model后
* python3 manage.py makemigrations sheet
* python3 manage.py migrate sheet
